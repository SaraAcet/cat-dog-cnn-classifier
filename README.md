# Kedi Köpek CNN Sınıflandırıcı 

Bu proje, **TensorFlow ve Keras** kullanarak kedi ve köpek görüntülerini sınıflandırmak için **Evrişimsel Sinir Ağı (CNN)** modeli uygular.

Model, açık bir veri seti ile eğitilmiş olup performansı **Accuracy (Doğruluk), Precision (Kesinlik), Recall (Duyarlılık) ve F1 Skoru** gibi metriklerle değerlendirilmiştir.

# Cat Dog CNN Classifier 

This project implements a **Convolutional Neural Network (CNN)** to classify images of cats and dogs using **TensorFlow and Keras**.

The model is trained on a public dataset and evaluates performance using metrics such as **Accuracy, Precision, Recall and F1 Score**.

---

## Project Overview

* Image classification using CNN
* Binary classification: **Cat vs Dog**
* Implemented in **Python**
* Developed and trained using **Google Colab**
* Framework: **TensorFlow / Keras**
